# AId

A set of tools to test IA models or Agents.

This is the frontend part of the project.

[Backend AId Models Prject](https://github.com/rnascunha/aid-backend-models)

## Instalation

### Requiriments

- NodeJS / NPM
- Git - to clone the porject

### Build and Run

```bash
# Clone the project and enter directory
$ git clone https://github.com/rnascunha/aid.git
$ cd aid

# Install dependecies
$ npm install
```

To run in development:
```bash
$ npm run dev
```
Open a browser and enter to http://localhost:3000.

To compile the project for production:
```bash
$ npm run build
```
The output files will be put at the `.next` directory.
